## 安装

确保环境中已经安装了以下软件

- [Node.js](http://nodejs.org/)
- [Git](http://git-scm.com/)

环境安装后使用下方命令完成安装

```bash
npm install hexo-cli -g
```

## 创建

```bash
hexo init <folder>
```

## 启动

```bash
hexo server
```

## 部署

```bash
hexo generate
hexo deploy
```
